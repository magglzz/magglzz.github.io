The Search Flow

Search is an important element in how we help our customers. We provide a robust online directory of senior care that users can search through to find information, photos, reviews, and costs.

Question
How can we simplify the search flow to include as few steps as possible and create a better search experience?
Goals
To significantly increase engagement, qualitative usability, and conversion on the search results pages.
Research

Identifying Problems & Defining Goals

The original spec outlined the following problems:

  • Current search results do not provide enough information about the communities to decide which ones are interesting.
  • Current search results give the user too many choices to evaluate and no method of trimming them down
  • Users donâ€™t always have a great idea of what level of care they need

We used the following points to define what would make a better search experience:

  • User gets results without sense of mental exertion
  • User has a list of communities that they want to explore beyond the website
